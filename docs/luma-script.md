# Luma Dream Machine — chapter clips (trial-friendly script)

Use Luma's **web UI** at https://lumalabs.ai. No API needed.

## Setup before you begin

1. Have the reference image ready to upload:
   `scener/reference-frame-520.png` (frame 520 from `AlmostPerfektFull.mp4` —
   shows the lifted black sports car with exploded parts in the dark workshop;
   this gives Luma the right car, the right lighting, the right atmosphere).
2. Open Luma Dream Machine → click "Create" or "New generation".

## Per-clip workflow (repeat 9 times)

For each clip below:

1. **Upload start frame:** click the image-upload icon → pick
   `scener/reference-frame-520.png`. This becomes the first frame of the clip.
2. **Set aspect ratio:** click the aspect-ratio dropdown → pick the value
   shown for the clip (16:9 or 9:16). This is the only manual UI step that
   matters for cost.
3. **Paste the prompt** into the prompt field.
4. **Set the title** (top of the generation panel, optional but recommended):
   matches the `Title:` field in the clip. Saves you from guessing later.
5. **Generate.** Wait 30 sec – 3 min depending on Luma queue.
6. **Download** the MP4 from Luma → rename it to the `Filename:` value before
   moving on. Save to `scener/`.

> **Trial credit budget:** ray-flash-2 (default on trial) ≈ 1 credit per
> 5-sec clip. 9 clips = ~9 credits of your 30. Run the 4 desktop chapter
> clips first — they're the essential ones; mobile clips are optional.
